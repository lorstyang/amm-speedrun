# AMM Lab (Uniswap v2)

用于学习与测试的单池 AMM 交互网页，基于 React + TypeScript + Vite。

##

- Milestone A: swap exact-in、曲线点位更新、spot/avg/slippage/fee 指标
- Milestone B: add/remove liquidity、LP 份额、累计手续费、history + undo/redo
- Milestone C: 预设场景、价格时间线、导出/导入状态

## 本地运行

```bash
npm install
npm run dev
```

## 运行测试

```bash
npm run test
```
